# Rof 



Rof — приложение, похожее на Stack Overflow.

### ⭐ Функционал:

- Система вопросов/ответов
- Система оценки вопроса/ответа
- Ответы от ChatGPT
- Глобальный Поиск
- Рекомендации, пагинация, фильтры и сортировка
- Профиль, редактирование профиля.
- Награды.
- Репутация и рейтинг
- Избранное
- Просмотры
- Топы авторов, статей
